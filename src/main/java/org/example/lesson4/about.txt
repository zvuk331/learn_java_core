Static вложенные (Nested) классы можно создать без экземпляра верхнего класс, например Car.Engine = new Car.Engine();
А внутренние классы (Inner) можно создать только с помощью экземпляра верхнего класса:
Car car = new Car();
Car.Engine engine = car.new Engine();

Local Inner Class - локальный класс, класс созданный внутри метода.

Anonymous class - анонимный класс, который используется для разового использования
